# DESAFIO - Trilha Java Intermediate (janeiro/2025) 

## Descrição
API de gestão financeira para clientes bancários que permite aos usuários o gerenciamento de suas finanças pessoais.

### Tecnologias
* Spring Boot 3
* Java 17
* MySQL

### Dependências
* Spring Web
* pring Boot DevTools
* Lombok
* Spring Data JPA
* Validation
* Flyway Migration
* MySQL Driver
* Spring Security
